\# SMS Spam Detection using Machine Learning



\## Description

This project implements an SMS spam detection system using machine learning techniques.



\## Dataset

SMS Spam Collection Dataset (Kaggle)



\## Technologies Used

\- Python

\- Pandas, NumPy

\- Scikit-learn

\- TF-IDF Vectorizer

\- Naive Bayes Classifier



\## Result

The model achieved good accuracy in classifying spam messages.
